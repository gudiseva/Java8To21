## GIT Commands

### Navigate to your local project directory
cd /path/to/your/local/project

### Initialize Git (skip if already initialized)
git init

### Add all project files to the staging area
git add .

### Create your initial local commit
git commit -m "Initial commit"

### Link your local project to the remote GitHub repository
git remote add origin https://github.com/gudiseva/Java8To21.git

### Rename your default local branch to main (recommended)
git branch -M main

### Upload your files to GitHub
git push -u origin main

### Pull existing files and merge histories (if required)
git pull origin main --allow-unrelated-histories

### Fix any conflicts if prompted, then push normally
git push -u origin main

### If you want to completely overwrite whatever is currently on GitHub with your local project files instead, use the force flag
git push -u -f origin main

### To fix all ignored files at once
git rm -r --cached .

git add .

git commit -m "Fix .gitignore and untrack excluded files"
